# ⚡ Démarrage Rapide

## En 3 étapes

### 1️⃣ Installer
```bash
npm install
```

### 2️⃣ Lancer
```bash
npm run dev
```

### 3️⃣ Accéder
Ouvrir http://localhost:3000

---

## 🚀 Déployer

### Vercel (1 minute)
```bash
# Installer Vercel CLI
npm i -g vercel

# Déployer
vercel
```

### Ou via Interface Web
1. Aller sur [vercel.com](https://vercel.com)
2. "New Project" → Importer depuis GitHub
3. Cliquer "Deploy"
4. C'est fait ! ✅
